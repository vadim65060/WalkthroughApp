# 🏠 Walkthrough — Обход квартир

Приложение для записи информации при обходе квартир.  
Создавайте дома, добавляйте жильцов, отмечайте «нет дома» и экспортируйте данные в CSV или JSON.

---

## ✨ Возможности

- 🏘️ **Управление домами** – добавление, редактирование и удаление домов.
- 🚪 **Обход квартир** – быстрый ввод данных с переключением шага +1/-1.
- 📱 **Автоопределение адреса** – GPS определяет ваш город, улицу и номер дома.
- 📞 **Умный ввод телефона** – валидация номеров с подстановкой кода города (мобильные и домашние).
- 📊 **Экспорт данных** – CSV и JSON в выбранную папку или буфер обмена.

---

## 🛠️ Технологии

- **Kotlin** + **Jetpack Compose** – современный UI.
- **Room** – локальное хранение данных.
- **Google Play Services Location** – определение геопозиции.
- **libphonenumber** – валидация телефонных номеров.
- **Gson** – работа с JSON.
